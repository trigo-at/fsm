    public addTransition(transition: object): * source

    Add a new transition to the state machine

    Params:

    NameTypeAttributeDescription
    transition object

    the name of the transition

    Suported syntax: (camelCasle, colon:case, snake_case, dash-case)
    myTransition, my:transition, my_transition, my-transition => will create function .myTransition()
    transition.name string

    the name of the transition
    transition.form string | object | Array<string> | Array<object>

    the states from which this transition can be triggered.

    Supported syntax:
    '*' => allow from any state 'state1' => allow only from state 'state1'
    ['state1', 'state2'] => allow transition from state1 & state2
    { state: ['s1', 's2'], substate: ['sub1', 'sub2'] } => allow transition from all states creates by calculating all
    permutations of the given states. 'state:s2|substate:sub1' 'state:s2|substate:sub1' 'state:s2|substate:sub2'
    transition.form string | object

    defines the state that is created after the transition

    Supported syntax:
    '*' => do not change source state
    'state1' => sets target state to 'state1'
    { state: 's1' } => sets the property "state" of the current state to "s1" eg. patches the existing state object representation
